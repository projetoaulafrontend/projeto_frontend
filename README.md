## ✨ Sobre o projeto

O **Flora & Folio** é um projeto front-end de e-commerce desenvolvido com foco em **cosméticos sustentáveis**, combinando estética moderna, organização visual e interações que tornam a navegação mais agradável.

A proposta da aplicação é apresentar produtos de forma clara e atrativa, permitindo ao usuário explorar o catálogo, visualizar detalhes, filtrar opções e interagir com um carrinho de compras, além de acessar uma página de contato para atendimento.

---

## 🎯 Objetivo

Este projeto foi criado para praticar e demonstrar conhecimentos em:

- estruturação de páginas com **HTML5**
- estilização moderna e responsiva com **CSS3**
- interatividade com **JavaScript**
- uso de componentes responsivos com **Bootstrap**
- organização de interface para projetos de **e-commerce**

---

## 🖥️ Funcionalidades

- **Página inicial** com banner em carrossel e navegação principal  
- **Busca de produtos** na interface  
- **Catálogo de produtos** com organização visual em cards  
- **Filtros por preço** e categorias de produto  
- **Visualização detalhada** em modais  
- **Carrinho de compras** com adicionar, remover e limpar itens  
- **Página de contato** com formulário e informações de atendimento  
- **Layout responsivo** para diferentes tamanhos de tela
